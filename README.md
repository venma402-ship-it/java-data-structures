# Java Data Structures Library

A simple implementation of fundamental data structures from scratch in Java.

## Implemented Data Structures

### Stack
Operations:
- push()
- pop()
- peek()
- size()
- display()

### Queue
Operations:
- enqueue()
- dequeue()
- display()

### Linked List
Operations:
- insert()
- delete()
- search()
- size()
- display()
